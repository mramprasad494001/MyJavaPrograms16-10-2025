## -: Inheritance :- (basic hum yha extends keyword ka use karenge)

## 4 Type of Inheritance

1. single level Inheritance -
  * jab do class ke bich relation Establise hota hai tab use* single level Inheritance kahte hai.

2. multy level Inheritance -
  * jab do se adhik class ke saath relation stablize karne ko multy level Inheritance kahte hai.

3. Hierarchical Inheritance -
  * jab parent class ko Multipule child class extends karte hai to use* Hierarchical Inheritance kahte hai. mager wah saare child class ek dusre se relation nhi rakhte hai.

4. Multiple Inheritance -
  * esme interface ka use karke parent class ko child class implement karta hai. kyu ki mutiple me posible nhi hai
